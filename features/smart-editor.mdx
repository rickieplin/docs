## AI Commands

Access these powerful commands by typing `/` in the editor:

<AccordionGroup>
  <Accordion title="/rewrite - Rewrite Selected Text">
    Intelligently rewrite any selected text while maintaining your intended meaning
    
    ```
    /rewrite make this more formal
    /rewrite simplify for general audience
    /rewrite add more detail
    ```
  </Accordion>
  
  <Accordion title="/expand - Expand Your Ideas">
    Elaborate on your ideas with additional context and details
    
    ```
    /expand add supporting arguments
    /expand include examples
    /expand provide more context
    ```
  </Accordion>
  
  <Accordion title="/summarize - Create Summaries">
    Generate concise summaries of selected text or entire sections
    
    ```
    /summarize in 3 bullet points
    /summarize as executive summary
    /summarize key takeaways
    ```
  </Accordion>
  
  <Accordion title="/translate - Translate Content">
    Translate your content into any supported language
    
    ```
    /translate to Spanish
    /translate to Simplified Chinese
    /translate to French (formal)
    ```
  </Accordion>
</AccordionGroup>

## Integration

SmartEditor Pro seamlessly integrates with your workflow:

```javascript
// Enable SmartEditor in your document
const editor = await docmind.editors.create({
  mode: 'smart',
  language: 'en',
  style: 'professional',
  features: {
    autoSuggest: true,
    grammarCheck: true,
    styleGuide: 'corporate'
  }
});

// Listen for suggestions
editor.on('suggestion', (suggestion) => {
  console.log('AI suggests:', suggestion.text);
});
```
